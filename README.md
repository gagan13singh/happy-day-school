## 🎨 About This Project

Happy Day School is a **production website** built for a real playschool in Delhi. The goal was simple: create a warm, welcoming digital presence that immediately communicates trust and joy to parents visiting for the first time.

Every design decision was made with one question in mind: *"Would a parent feel confident enrolling their child here?"*

---

## ✨ What's on the Website

### 🏫 About the School
A heartfelt introduction to the school — its mission, values, and the nurturing environment it provides for young learners.

### 📚 Programmes & Curriculum
Clear breakdown of age groups, learning approaches, and the activities that make every day at Happy Day School an adventure.

### 🎠 Gallery
A visual showcase of school life — classrooms, activities, events, and happy faces.

### 📞 Contact & Admissions
Parents can easily find the school's location, phone number, and enquiry form — reducing friction for new admissions.

### 📱 Fully Responsive
Designed mobile-first. Most parents will visit on their phones — the site looks perfect on every screen size.

## 🚀 Running Locally

```bash
# Clone
git clone https://github.com/gagan13singh/happy-day-school.git
cd happy-day-school

# Install
npm install

# Run
npm run dev
```

Open `http://localhost:5173` in your browser.

## 🎯 Design Goals

```
✓  Warm, playful colour palette (oranges, yellows, pinks)
✓  Easy navigation for non-technical parents

✓  Fast load times (Vercel CDN)

```

---

## 📸 Design Philosophy

Playschool websites carry a unique responsibility — they speak to **parents, not children**. The design balances:

- **Playfulness** — colours and shapes that feel joyful and safe
- **Professionalism** — clean layout, readable content, clear info hierarchy
- **Warmth** — language and visuals that feel personal, not corporate
